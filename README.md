# ebiutil

Utility library for Ebitengine (Go game engine)

## Packages
- `geom` - Geometry types (Point, Rect)
- `ui` - UI components (Button)
- `utils` - Utility functions
- `sprites` - Sprite base struct
- `entities` - Game entities (Ball)

## Install
go get github.com/song9063/ebiutil
